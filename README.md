# Grade-Calculator-Program
 Project Description
This project is a simple Student Grade Calculator written in Python.
The program allows users to enter student names and grades, store them, and perform basic calculations such as:
Viewing all students
Calculating the average grade
Finding the highest and lowest grades
The goal of this project is to demonstrate basic Python programming concepts such as lists, functions, loops, and file handling.
⚙️ Features
Add new students and grades
View all stored student records
Calculate average grade
Find highest and lowest grades
Save data to a CSV file
Load saved data automatically when the program starts
🛠️ Requirements
To run this program, you need:
Python 3.x installed
A code editor (recommended: Visual Studio Code)
How to Run the Program
Install Python 3 on your computer
Download or clone this repository:
git clone https://github.com/your-username/student-grade-calculator.git
Open the project folder
Run the program using:
python grade_calculator.py
📂 File Structure
grade_calculator.py → Main program file
grades.csv → Stores student data (created automatically)
 How It Works
The program uses a list to store student data
Each student is stored as:
[name, grade]
Data is saved using the CSV file format, so it is not lost when the program closes
A menu system allows the user to interact with the program
